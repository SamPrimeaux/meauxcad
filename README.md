## 🚀 Overview

MeauxCAD Studio is a high-performance, desktop-class IDE built entirely on the Cloudflare ecosystem (Workers, D1, R2). It is designed to be a "living" environment where an AI Agent and a Human Developer work in perfect synchrony across code, 3D assets, and terminal operations.

**Live Demo:** [meauxcad.meauxbility.workers.dev](https://meauxcad.meauxbility.workers.dev)

---

## ✨ Key Features

### 💻 Monaco Editor (Pro Grade)
- **Real File System Access:** Full read/write/save capabilities via the File System Access API. Your changes persist directly to your local disk.
- **Dirty State Tracking:** Visual indicators (`●`) and smart "Discard" logic to manage unsaved changes.
- **Integrated Diff View:** Side-by-side comparison of local vs. buffered changes.
- **Multilingual Support:** Syntax highlighting and intelligent defaults for 20+ languages including TypeScript, Python, Rust, Go, SQL, and more.

### 🐚 XTerm Terminal & Agent Bridge
- **Interactive Shell:** A full-featured terminal emulator with Solarized Dark aesthetics.
- **Agent Command Injection:** The AI Agent can programmatically write to the terminal and execute commands ($ Run in Terminal).
- **Tabbed Drawer:** Toggle between `Terminal`, `Output`, and `Problems` (with real-time error/warning counts).

### 🤖 Antigravity AI Agent
- **Context-Aware Coding:** The agent understands your active file, project structure, and even your terminal output.
- **Artifact System:** Code blocks generated by the agent can be instantly injected into the editor or run in the terminal.
- **MCP Tool Integration:** Real-time management of Model Context Protocol (MCP) tools directly within the IDE.

### 🧊 3D & Web Preview
- **Voxel Engine:** A built-in 3D engine for voxel-based design and experimentation.
- **GLB Viewport:** Native rendering of 3D assets (`.glb`) with high-fidelity lighting.
- **In-IDE Browser:** Test your web applications without ever leaving the workspace.

## 🛠 Tech Stack

- **Frontend:** React, Vite, TailwindCSS (for shells), Vanilla CSS (for core IDE logic).
- **Editor:** Monaco Editor (`monaco-editor`).
- **Terminal:** XTerm.js.
- **Backend:** Cloudflare Workers (Edge Computing).
- **Database:** Cloudflare D1 (SQL).
- **Storage:** Cloudflare R2 (Object Storage).
- **AI Integration:** Google Gemini API (via Antigravity).

## 🚦 Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) (v18+)
- [Wrangler CLI](https://developers.cloudflare.com/workers/wrangler/install-and-update/) (for deployment)

### Local Development
1. **Clone the repo:**
   ```bash
   git clone https://github.com/SamPrimeaux/meauxcad.git
   cd meauxcad
   ```
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Configure environment:**
   Create a `.env.local` and add your `GEMINI_API_KEY`.
4. **Run the dev server:**
   ```bash
   npm run dev
   ```

### Deployment
This project is optimized for Cloudflare Workers.
```bash
npm run deploy
```

## 💎 Project Structure

- `/components`: Modular IDE components (MonacoEditorView, XTermShell, ChatAssistant).
- `/scripts`: Deployment and build automation scripts.
- `/utils`: Helper functions for file system and API interactions.
- `worker.ts`: The core Cloudflare Worker handling API routing and DB persistence.
- `App.tsx`: The IDE orchestration layer.
